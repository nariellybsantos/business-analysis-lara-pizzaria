#  Business Analysis – Lara Pizzaria Delivery

##  Sobre o Projeto

Este repositório reúne toda a documentação de Análise de Negócios desenvolvida para o projeto **Lara Pizzaria Delivery**.

O objetivo é compreender o negócio, levantar requisitos e documentar todas as etapas que antecedem o desenvolvimento de um sistema web.

Este projeto faz parte do meu portfólio profissional em Engenharia de Software.

---

##  Objetivos

- Compreender o modelo de negócio da empresa
- Identificar oportunidades de melhoria
- Levantar requisitos funcionais e não funcionais
- Documentar regras de negócio
- Produzir documentação técnica
- Servir como base para o desenvolvimento do sistema
